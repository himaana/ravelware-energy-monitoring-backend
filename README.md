# Energy Monitoring Backend

Backend service for energy monitoring dashboard using:

-   Node.js
-   MQTT
-   InfluxDB

## Features

-   Receive energy data via MQTT
-   Store time-series data in InfluxDB
-   Provide REST API for Dashboard
-   Calculate daily usage and cost
-   Detect panel online/offline status

## How to Run

1. Setup InfluxDB
2. Configure .env
3. npm install
4. npm start
